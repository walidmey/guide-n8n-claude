# Vue d'ensemble

{% hint style="info" %}
Cette sous-partie pose le cadre pratique de la mise en route. Vous allez voir ce que vous aurez installé, connecté et validé avant de passer aux workflows détaillés.
{% endhint %}

Cette partie est entièrement pratique. À la fin, vous aurez :

* n8n installé et opérationnel
* Claude connecté via l'API Anthropic
* Un premier workflow fonctionnel qui appelle Claude et traite sa réponse

Temps estimé : 1 à 2 heures.

{% hint style="success" %}
Résultat attendu : en fin de partie, vous pourrez déclencher un workflow n8n qui envoie un texte à Claude, récupère sa réponse, puis l’utilise automatiquement dans une étape suivante.
{% endhint %}

{% hint style="info" %}
Vous aurez besoin d'une clé API Anthropic (payante, environ 5 à 20 € par mois selon votre usage) et soit d'un compte n8n Cloud (gratuit jusqu'à un certain volume) soit d'un serveur pour l'auto-hébergement.
{% endhint %}
