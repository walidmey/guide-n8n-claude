# Partie 4 — Aller plus loin

Vous avez maintenant cinq workflows opérationnels. Cette partie est pour ceux qui veulent aller au-delà des automatisations simples.

Deux chapitres, deux sujets distincts :

**Agents chaînés** — comment faire travailler plusieurs IAs ensemble dans un même workflow, où une décision de Claude déclenche une autre action de Claude. Ce n'est plus de l'automatisation linéaire : c'est de l'orchestration.

**Mise en production** — ce qui change quand un workflow passe de "ça marche sur mon poste" à "ça tourne en production depuis 3 mois sans que j'y touche". Monitoring, gestion des erreurs, coûts, sécurité.

Ces deux chapitres sont indépendants — lisez celui qui correspond à votre étape actuelle.
