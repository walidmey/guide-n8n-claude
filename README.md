# n8n + Claude : automatiser ses workflows sans coder

**Automatiser des tâches complexes avec l'IA. Sans écrire une seule ligne de code.**

n8n est un outil d'automatisation visuel. Claude est un modèle de langage capable de lire, analyser, rédiger et décider. Ensemble, ils permettent de construire des workflows intelligents que ni l'un ni l'autre ne pourrait faire seul.

Ce guide vous montre comment les combiner — de l'installation à la mise en production — avec des exemples concrets et réutilisables.

## À qui s'adresse ce guide ?

À vous, si :
- Vous passez des heures chaque semaine sur des tâches répétitives (emails, résumés, rapports, tri d'informations).
- Vous avez entendu parler de l'automatisation mais vous n'êtes pas développeur.
- Vous utilisez déjà Claude ou un autre LLM mais vous ne savez pas comment l'intégrer dans vos processus.
- Vous voulez des résultats concrets, pas une conférence sur "l'avenir du travail".

{% hint style="info" %}
Aucune compétence en programmation n'est requise. Si vous savez créer un tableau Excel et copier-coller un JSON, vous pouvez suivre ce guide.
{% endhint %}

## Comment utiliser ce guide

Lisez-le dans l'ordre si vous partez de zéro. Sautez à la Partie 3 si vous connaissez déjà n8n et voulez des workflows à réutiliser directement.

Chaque chapitre est autonome. Chaque workflow est complet — avec les étapes, les paramètres, et les variations possibles.
