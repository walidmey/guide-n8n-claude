# Partie 3 — 10 workflows prêts à l'emploi

Cette partie est le cœur du guide. Cinq workflows complets, immédiatement réutilisables. Chacun est documenté avec :

- **Le problème qu'il résout** — pour valider si c'est votre cas
- **Les nœuds nécessaires** — pour vérifier que vous avez les accès
- **Le schéma du workflow** — la séquence complète
- **Les paramètres détaillés** — y compris les prompts
- **Les variantes** — pour adapter à votre contexte

Choisissez le workflow le plus proche de votre situation, adaptez les paramètres, et activez.

{% hint style="info" %}
Tous ces workflows ont été testés avec des données réelles. Les temps d'exécution indiqués sont des estimations pour des entrées de taille standard.
{% endhint %}
